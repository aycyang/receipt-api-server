/login

Starts OAuth authorization code flow. After this is complete, a client application will be allowed to make authenticated HTTP requests to Receipt API Server by calling `fetch()` with the option `{credentials: 'include'}`.

  • redirect_uri: URL? - The URL to redirect back to after authentication is complete. Must be a *.recurse.com subdomain.

/text

Print text to the printer.
